## 策略参数

### 动态趋势策略（4周期模型）

每个持仓股票同时运行4套策略，互为参照：

| 周期 | 参考K线 | 适用场景 | 仓位权重 |
|------|---------|---------|---------|
| **3天** | 短线极趋势 | 追强势反弹、快进快出 | 灵活 |
| **12天** | 短期趋势 | 做波段、做T参考 | 主要 |
| **26天** | 中期趋势 | 波段操作核心参考 | 底仓 |
| **长期** | 年线方向 | 趋势判断、仓位决策 | 战略 |

---

### 各周期策略详情

#### 📅 3天策略（超短线）
- **参考周期**：最近3个交易日K线
- **核心指标**：RSI(6)、成交量变化、3日均线
- **操作频率**：每日可操作
- **止盈止损**：±2% 严格止损
- **适用信号**：
  - 低吸：价格回踩3日均线 + RSI(6) < 40
  - 高抛：价格偏离3日均线>+2% + RSI(6) > 60
- **仓位建议**：不超过总持仓的20%

#### 📅 12天策略（短期波段）
- **参考周期**：最近12个交易日K线
- **核心指标**：RSI(12)、MACD(12,26,9)、12日均线
- **操作频率**：每周1-3次
- **止盈止损**：±3%
- **适用信号**：
  - 买入：RSI(12) < 50 + MACD金叉 + 缩量回调至12日线
  - 卖出：RSI(12) > 65 + MACD死叉 + 放量滞涨
- **仓位建议**：总持仓的30-40%

#### 📅 26天策略（中期波段）
- **参考周期**：最近26个交易日K线
- **核心指标**：RSI(14)、MACD、布林带(26,2)、26日均线
- **操作频率**：每月2-4次
- **止盈止损**：±5%
- **适用信号**：
  - 买入：价格回调至26日均线支撑 + RSI(14) < 45
  - 卖出：价格触及布林上轨 + RSI(14) > 70
- **仓位建议**：总持仓的30-40%

#### 📈 长期策略（趋势判断）
- **参考周期**：55日均线（季线）、年线（250日）
- **核心指标**：55日均线方向、250日均线位置、RSI(55)
- **操作频率**：每月1次检视
- **止盈止损**：±8%
- **适用信号**：
  - 买入：55日均线向上 + 价格站上年线 + RSI(55) > 50
  - 卖出：55日均线向下 + 价格跌破年线
- **仓位建议**：底仓持有，40-50%

## 任务追踪系统

### 统一入口：stock_trading.py
所有股票交易功能通过单一入口调用：
```bash
python3 stock_trading.py [check|signal|review|brief|init|all]
  check  - 盘前检查（6模块）
  signal - 交易信号报告（14:30）
  review - 收盘复盘（15:05）
  brief  - 每日简报
  init   - 初始化数据库和任务
  all    - 初始化+简报
```

### 数据库
- `trading_task.db` — 任务表/交易日追踪/执行日志/策略运行记录
- `trading.db` — 持仓/交易记录/每日检查日志
