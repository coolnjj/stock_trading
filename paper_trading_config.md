## 三、一夜持仓法（3成）
### 选股范围
- **排除**：ST股、*ST股、科创板（688开头）
- **选股池**：沪深A股（000/001/002/600/601开头）

### 止盈止损指标
| 指标 | 参数 |
|------|------|
| **买入时间** | 14:30-14:50（尾盘观察） |
| **卖出时间** | 09:25-09:35（次日早盘） |
| **止盈** | +2%以上尾盘不板则了结 |
| **止损** | -2% |
| **最大单次亏损** | 总资金×-1% |

### 一夜持仓选股逻辑
```
1. 涨幅2%-5%的股票（避免追高）
2. 量比 > 1.2（资金活跃）
3. 流通市值 50-500亿（避免盘子过小/过大）
4. 排除涨停板（买不进去）
5. 尾盘有资金明显流入迹象
6. 次日开盘若低开-1%以上视情况补仓或止损
```

### 风险控制
- 单票不超过总资金5%
- 同时最多持仓5只
- 连续亏损3次自动暂停2天

## 四、龙头热点追踪（3成）
### 选股范围
- 排除：ST、*ST、科创板
- 重点：涨停板、强势板块龙头

### 止盈止损指标
| 指标 | 参数 |
|------|------|
| **追涨买入** | 涨幅>5%+放量+突破前高 |
| **止损** | -3%（跌破当日涨幅50%即出） |
| **止盈** | +5%起步，不板即出 |
| **最大单次亏损** | 总资金×-1.5% |
| **单票上限** | 总资金5% |

### 龙头追踪规则
```
1. 每日复盘找出当日最强板块（涨幅>3%的概念板块）
2. 选取板块内第一个涨停且封板坚定的龙头
3. 次日高开>+3%且放量可追入
4. 买入后亏损-3%立即止损
5. 若买入后次日涨停则持有观察，第三天不板即出
6. 每天最多操作1只，每波最多连做3天必须休息1天
```

### 热点识别规则
```
1. 涨幅>3%的概念/行业板块
2. 板块内涨停数>=3只
3. 板块资金净流入为正
4. 龙头股特征：率先涨停、封板早、抛压小
```

## 五、后备子弹（1成）
### 动用条件
| 场景 | 操作 |
|------|------|
| 某仓位亏损>-3% | 优先补仓至原位，不够再用子弹 |
| 某仓位盈利>+5% | 可转出盈利部分至子弹 |
| 全面大跌>-5% | 可分批补仓跌幅最大的仓位 |
| 全面大涨>+5% | 可将子弹转出 |

### 平衡机制
- 每周五收盘后检查一次 3:3:3:1 比例
- 偏离超过15%则触发再平衡
- 子弹不得低于总资金5%（保留底线）
